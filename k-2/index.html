    • Cybersecurity: This class teaches students how to protect themselves and others from cyber threats and attacks. Students will learn about cybersecurity concepts such as encryption, authentication, firewalls, malware, phishing, and hacking. Cybersecurity is a vital and challenging way to learn computer security and ethics skills.
    • E-sports: This class teaches students how to play video games competitively and professionally. Students will learn about e-sports genres such as first-person shooters, fighting games, sports games, and multiplayer online battle arenas. E-sports are a exciting and social way to learn gaming and teamwork skills.
    • Computer Hardware and Repair: This class teaches students how to identify, assemble, disassemble, and troubleshoot different parts of a computer system. Students will learn about hardware components such as CPU, RAM, motherboard, hard drive, power supply, and peripherals. Computer hardware and repair is a practical skill for understanding and maintaining computer systems.
    • Robotics (FLL/FTC/FRC): (First Lego League), ftc (First Tech Challenge), frc (First Robotics Competition) are different levels of robotics competitions for students of all ages. This class teaches students how to design, build, program, and control robots using various materials and platforms. Students will learn about robotics concepts such as sensors, actuators, motors, gears, algorithms, and logic. Robotics is a fun and creative way to explore science, technology, engineering, and math (STEM) fields5. Moreover, these competitions foster teamwork, problem-solving, innovation, and sportsmanship among students.
    • Minecraft Modding: This class teaches students how to create their own custom modifications (mods) and game designs for the popular sandbox game Minecraft. Students will learn how to use coding tools such as Java and Python to add new features, items, blocks, creatures, & worlds to the game. Minecraft modding is a fun and engaging way to learn coding and game development skills.
    • Digital Arts and 3D Animation: This class teaches students how to create digital artworks and animations using various software tools such as Photoshop, Illustrator, Blender, and Maya. Students will learn how to draw, paint, edit, and animate 2D and 3D images and models. Digital arts and 3D animation is a artistic and expressive way to learn graphic design and visual communication skills.
    • Roblox Design and Code: This class teaches students how to create their own games and experiences for the popular online platform Roblox. Students will learn how to use coding tools such as Lua and Roblox Studio to design game elements such as environments, characters, scripts, and GUIs. Roblox design and code is a fun and social way to learn coding and game development skills.
    • Career Exploration: This class helps students discover their interests, passions, and goals for the future in the fields of science, technology, engineering, and math (STEM). Students will learn about different STEM careers and opportunities in various sectors and industries, such as medicine, research, robotics, biochemistry, aeronautics, and more. Students will also learn how to prepare for these careers through education, training, and networking. 
    • E-sports: This class teaches students how to play video games competitively and professionally. Students will learn about e-sports genres such as first-person shooters, fighting games, sports games, and multiplayer online battle arenas. E-sports are a exciting and social way to learn gaming and teamwork skills.
    • Keyboarding: This class teaches students how to type faster, easier, and better on a keyboard. Students will learn proper hand posture, accuracy, and speed through engaging lessons, games, and tests. Keyboarding is an essential skill for writing, communicating, and succeeding in school and beyond.
    • Digital Citizenship: This class teaches students how to be responsible, respectful, and safe online. Students will learn about online behavior, privacy, security, cyberbullying, digital footprint, and media literacy. Digital citizenship helps students navigate the digital world with confidence and awareness.
    • Computer Literacy: This class teaches students the basics of using a computer, such as operating systems, applications, files, folders, and internet browsers. Students will also learn how to use common software tools such as word processors, spreadsheets, presentations, and email. Computer literacy is a foundational skill for learning and working with technology.
